== Description ==
T7ix Mercury Lite is a modern, lightweight, and flexible WordPress theme built with full support for Gutenberg blocks.
Designed for high performance and seamless customization, the theme features built-in block patterns, advanced theme options, and optimized styles for a smooth user experience.
Ideal for businesses, portfolios, and content-driven websites.

== Installation ==
1. Download the theme ZIP file.
2. Go to **Appearance > Themes** in your WordPress dashboard.
3. Click **Add New** and then **Upload Theme**.
4. Select the ZIP file and click **Install Now**.
5. Activate the theme after installation.
